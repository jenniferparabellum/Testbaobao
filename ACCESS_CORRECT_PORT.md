# ⚠️ 重要：访问正确的端口

## 问题
你访问了 **http://localhost:8081/**，但这是**错误的端口**！

## ✅ 正确的访问地址

**请访问：http://localhost:19006**

**不是** http://localhost:8081

## 端口说明

- **8081** = Metro Bundler（开发服务器，用于移动端）
- **19006** = Web应用（这是你要访问的网页）

## 如何确认正确的地址

在启动服务器的终端中，你应该看到：
```
› Metro waiting on exp://192.168.x.x:8081
› Web is waiting on http://localhost:19006  ← 访问这个！
```

## 访问步骤

1. **关闭当前浏览器标签页**（8081的那个空白页面）
2. **打开新标签页**
3. **在地址栏输入**：`http://localhost:19006`
4. **按回车访问**

## 如果19006也显示空白

### 方法1: 检查浏览器控制台
1. 按 **F12** 打开开发者工具
2. 查看 **Console** 标签中的错误信息
3. 告诉我看到了什么错误

### 方法2: 清除缓存刷新
- macOS: **Cmd+Shift+R**
- Windows/Linux: **Ctrl+Shift+R**

### 方法3: 等待服务器完全启动
Expo首次启动Web可能需要30-60秒，请等待直到看到 "Web is waiting on http://localhost:19006" 消息。

## 正确的启动流程

1. 在终端运行 `npm run web`
2. 等待看到 "Web is waiting on http://localhost:19006"
3. 访问 **http://localhost:19006**（不是8081）

## 测试

访问正确的地址后，你应该看到：
- ✅ "宝宝记录" 标题
- ✅ 三个功能卡片
- ✅ "查看历史记录" 按钮

如果还是有问题，请查看浏览器控制台的错误信息。
