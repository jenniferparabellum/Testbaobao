## 如果遇到问题

### 问题1: 端口被占用
如果19006端口被占用，Expo会自动使用其他端口，查看终端输出中的实际地址。

### 问题2: 浏览器没有自动打开
手动在浏览器中访问：**http://localhost:19006**

### 问题3: 页面空白或加载错误
1. 检查浏览器控制台是否有错误（按F12打开开发者工具）
2. 清除浏览器缓存并刷新页面（Cmd+Shift+R 或 Ctrl+Shift+R）
3. 查看终端输出中的错误信息

### 问题4: 需要停止服务器
在终端中按 **`Ctrl + C`** 停止服务器

## 检查服务器状态

在终端中运行：
```bash
lsof -i :19006
```

如果看到输出，说明服务器正在运行。
