### 方式1: 浏览器自动打开
应用会在**默认浏览器中自动打开**（通常在启动后几秒内）

如果浏览器没有自动打开，请手动访问：

**http://localhost:19006**

## 测试功能

### ✅ 1. 记录功能
- **记录吃饭** - 点击"吃饭"卡片的"记录"按钮
- **记录睡觉** - 点击"睡觉"卡片的"记录"按钮
- **记录换尿布** - 点击"换尿布"卡片的"记录"按钮
- 检查是否显示"记录已保存！"提示

### ✅ 2. 查看历史记录
- 点击"查看历史记录"按钮
- 查看记录是否按日期分组显示
- 查看记录时间是否正确显示

### ✅ 3. 删除记录
- 在历史记录页面点击记录右侧的"删除"按钮
- 确认记录被删除
- 返回主页面验证

### ✅ 4. UI测试
- 检查页面布局是否正常
- 检查图标和颜色显示
- 测试按钮点击响应
- 测试页面滚动

## 如果遇到问题

### 问题1: 无法访问 http://localhost:19006
1. 检查服务器是否正在运行
2. 查看终端输出是否有错误
3. 尝试刷新页面（Cmd+R 或 Ctrl+R）

### 问题2: 页面空白或加载错误
1. 打开浏览器开发者工具（按F12）
2. 查看Console标签中的错误信息
3. 清除浏览器缓存并刷新（Cmd+Shift+R 或 Ctrl+Shift+R）

### 问题3: 端口被占用
如果19006端口被占用，Expo会自动使用其他端口（如19007），查看终端输出中的实际地址。

### 问题4: 需要停止服务器
在终端中按 **`Ctrl + C`** 停止服务器

## 检查服务器状态

在终端中运行：
```bash
lsof -i :19006
```

如果看到输出，说明服务器正在运行。
