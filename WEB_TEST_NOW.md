## ❌ 如果遇到问题

### 问题1: 页面无法访问（ERR_CONNECTION_REFUSED）
1. **等待60秒**让服务器完全启动
2. **检查浏览器地址栏**是否正确（http://localhost:19006）
3. **尝试刷新页面**（Cmd+R 或 Ctrl+R）

### 问题2: 页面空白
1. **打开浏览器开发者工具**（按F12）
2. **查看Console标签**中的错误信息
3. **查看Network标签**，检查资源是否加载
4. **清除浏览器缓存**（Cmd+Shift+R 或 Ctrl+Shift+R）

### 问题3: 需要查看服务器输出
如果你想在前台运行服务器以查看详细输出，请在终端中运行：

```bash
cd /Users/parabellum/Desktop/Test
export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
nvm use 18
npm run web
```

这样可以看到完整的启动过程和任何错误信息。

### 问题4: 需要停止服务器
```bash
pkill -f "expo start"
```
