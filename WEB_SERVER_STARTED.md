# 🚀 Web服务器已启动

## 服务器信息

Web开发服务器正在后台运行中。

## 访问方式

应用应该会在**默认浏览器中自动打开**，如果没有自动打开，请手动访问：

**http://localhost:19006**

## 如果浏览器没有自动打开

1. 打开你的浏览器（Chrome、Safari、Firefox等）
2. 在地址栏输入：`http://localhost:19006`
3. 按回车键访问

在浏览器中测试以下功能：

1. ✅ **记录吃饭** - 点击"吃饭"卡片的"记录"按钮

## 停止服务器

如果需要停止服务器，在终端中按：

**`Ctrl + C`**

或者找到进程并终止：

```bash
lsof -ti:19006 | xargs kill
```

## 常见问题

### 端口被占用
如果19006端口被占用，服务器会自动使用其他端口（如19007），查看终端输出中的实际地址。

### 页面无法加载
1. 检查服务器是否正在运行
2. 检查浏览器控制台是否有错误（按F12打开开发者工具）
3. 尝试刷新页面（Cmd+R 或 Ctrl+R）

### 需要重新启动
如果修改了代码，保存后页面会自动刷新（热重载）。如果需要完全重启服务器：

```bash
# 停止当前服务器（Ctrl+C），然后重新运行
npm run web
```
