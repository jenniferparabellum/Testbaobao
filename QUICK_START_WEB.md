# 🚀 快速启动Web服务器

## Web服务器已启动！

服务器正在后台运行中。

## 📱 访问应用

### 方式1: 浏览器自动打开
应用会在**默认浏览器中自动打开**（通常在启动后10-15秒内）

### 方式2: 手动访问
如果浏览器没有自动打开，请手动访问：

**http://localhost:19006**

或者如果端口被占用，Expo可能使用了其他端口（如19007），请查看终端输出。

## ⏱️ 如果服务器还在启动

Expo服务器启动通常需要10-20秒。请稍等片刻，然后：

1. **检查浏览器是否自动打开**
2. **手动访问** http://localhost:19006
3. **查看终端输出**（如果有终端窗口显示服务器日志）

## 🧪 测试功能

### ✅ 1. 记录功能
- 点击"吃饭"卡片的"记录"按钮 → 应该显示"记录已保存！"
- 点击"睡觉"卡片的"记录"按钮
- 点击"换尿布"卡片的"记录"按钮

### ✅ 2. 查看历史记录
- 点击"查看历史记录"按钮
- 查看记录是否按日期分组
- 查看记录时间是否正确

### ✅ 3. 删除记录
- 在历史记录页面点击"删除"按钮
- 确认记录被删除

## ❌ 如果遇到问题

### 问题1: 页面无法访问
1. 等待10-20秒让服务器完全启动
2. 检查浏览器地址栏是否正确（http://localhost:19006）
3. 尝试刷新页面（Cmd+R 或 Ctrl+R）

### 问题2: 页面空白
1. 打开浏览器开发者工具（按F12）
2. 查看Console标签中的错误信息
3. 清除浏览器缓存（Cmd+Shift+R 或 Ctrl+Shift+R）

### 问题3: 需要查看服务器日志
如果服务器在后台运行，可以在终端中查看进程：
```bash
ps aux | grep expo
```

### 问题4: 需要停止服务器
```bash
lsof -ti:19006 | xargs kill
# 或
pkill -f "expo start"
```
